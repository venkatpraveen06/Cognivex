# Cognivex

Cognivex is a modern technology startup focused on AI tools, automation solutions, web development, and digital innovation.

This project is a premium enterprise-grade website built using HTML, CSS, and JavaScript, inspired by the user experience and sophistication of leading technology companies. It features a futuristic design, responsive layout, smooth animations, interactive sections, and a professional startup presentation.

## Features

- Modern enterprise UI/UX
- Responsive design
- Smooth scroll animations
- Interactive service sections
- Glassmorphism effects
- Premium landing page
- WhatsApp integration
- AI-inspired branding
- Fast and lightweight

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Mission

To build intelligent digital solutions through AI, automation, and modern web technologies.
